# 5分钟快速部署指南 ⚡

## 🎯 目标
让你的网站在 5 分钟内上线，获得一个可以分享的链接！

---

## 📝 超简化步骤

### 1️⃣ 注册 Gitee（1分钟）
👉 打开：https://gitee.com
- 点击"注册"
- 填写邮箱、用户名、密码
- 验证邮箱

### 2️⃣ 创建仓库（1分钟）
- 点击右上角 "+" → "新建仓库"
- 仓库名：`website`
- 选择"公开"
- 点击"创建"

### 3️⃣ 上传文件（1分钟）
- 点击"上传文件"
- 拖拽 `index.html` 文件
- 点击"提交"

### 4️⃣ 启用 Pages（1分钟）
- 点击"服务" → "Gitee Pages"
- 点击"启动"
- 选择 `master` 分支
- 点击"启动"

### 5️⃣ 获取链接（立即）
✅ 复制显示的链接：
```
https://你的用户名.gitee.io/website
```

---

## 🎉 完成！

**把这个链接发给任何人，他们都可以在浏览器中打开你的网站！**

## 📱 测试方法

1. 复制链接
2. 在浏览器新标签页打开
3. 在手机浏览器打开
4. 发给朋友测试

---

## 🔄 如何更新？

1. 进入 Gitee 仓库
2. 点击 `index.html` → "编辑"
3. 修改代码
4. 点击"提交"
5. 回到"服务" → "Gitee Pages" → 点击"更新"

---

## ❓ 遇到问题？

### 问题1：显示 404
- 等待 1-2 分钟
- 确保文件名是 `index.html`
- 确保仓库是"公开"

### 问题2：需要实名认证
- 在个人设置中完成实名认证
- 使用身份证或支付宝认证

### 问题3：修改后没更新
- 在 Gitee Pages 点击"更新"
- 清除浏览器缓存（Ctrl + F5）

## 🎊 其他快速方案

### 方案A：GitHub Pages（国际化）
1. 注册 https://github.com
2. 创建仓库（名称：`username.github.io`）
3. 上传 `index.html`
4. 访问：`https://username.github.io`

### 方案B：Vercel（最快）
1. 访问 https://vercel.com
2. 用 GitHub 登录
3. 导入仓库
4. 自动部署完成

### 方案C：Netlify（拖拽部署）
1. 访问 https://netlify.com
2. 注册登录
3. 拖拽文件夹到页面
4. 立即获得链接
